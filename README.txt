Install Virtualbox
https://www.virtualbox.org/wiki/Downloads

Install Vagrant
https://www.vagrantup.com/downloads.html

Clone GitHub repository
https://github.com/udacity/fullstack-nanodegree-vm

change directory tournament directory in Git Bash
-->"vagrant up" to intialize virtual machine for the first time
-->type "vagrant ssh" to log into machine
-->"cd /vagrant" and then "cd tournament" to change into 
directory of your files
--"psql tournament" to connect to tournament
-->"\i tournament.sql" to execute sql commands

go to https://www.postgresql.org/docs/8.2/static/sql-commands.html
for any other commands you might need. 
GOOD LUCK!
